HTML source element

The source tag is used to specify multiple resources for a media (i.e., a picture, video, or audio tag)

The source tag's required attributes and usage depends on the type of its parent container

source tag within a picture tag

When used within a picture tag, the srcset attribute is required. The media attribute can be used to specify the conditions for which a source should be used

source tag within an audio/video tag

When used within an audio/video tag, the src attribute is required. If the type attribute is specified, the browser will use the first source with a supported type
