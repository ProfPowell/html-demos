HTML picture element

The picture tag acts as a container for multiple different sources for a single image. It can have any number of source tags, but must have a single img tag at the very end.

The picture tag is used to select different source images for different contexts. Like the img tag, the picture tag can implement resolution switching.

However, it's best to reserve the picture tag for more complex cases that do not work with the img tag. For example, in addition to resolution switching, we may want to crop an image depending on the width of the browser

The src attribute of the img tag is used only if none of the source tags could be used or if the browser does not support the picture tag
