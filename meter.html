How to use html tag <meter>

Description:

The meter tag is used to show completion status as a loading bar, it has many attributes that defines its appearence:
min, max, low, high: are used to define range of values in the bar
value, optimum: represents the current and optimum values
*Color of bar depending on the numeric relation between attribute values, if value falls into optimum range, it is
shown as green, and orange if it falls close to optimum, red if far from optimum
