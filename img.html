HTML img element

The image tag displays an image. It requires a src attribute to provide the resource image and an alt attribute that describes the image.

It will display the alt attribute's value if the image cannot be loaded.

Images can be animated.

Images can lead to a link. (Click the below image)

You can specify the srcset attribute to load different images depending on the dimensions of the browser. This is useful when you want to switch the resolution of an image based on the size of a browser. Try resizing your browser and see how the below image changes.
